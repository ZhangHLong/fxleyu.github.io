```sql
SELECT
    c.sku_id, c.comment_num, c.bad_comment_rate
FROM
    jdata.action a,
    jdata.comment c
WHERE
    a.type = 4 AND a.sku_id = c.sku_id;
```

加索引

```sql
ALTER TABLE `jdata`.`action`
ADD INDEX `action_pid` (`sku_id` ASC);
```
